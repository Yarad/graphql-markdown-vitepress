# Demo: graphql-markdown-vitepress

A minimal VitePress site that documents the included `schema-example.json` using `graphql-markdown-vitepress`.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Then open the dev server URL and use the sidebar to browse the API.

## Build

```bash
npm run build
npm run preview
```

## Styling

The generated HTML uses CSS classes styled in [`.vitepress/theme/custom.css`](./.vitepress/theme/custom.css). When using a custom `css` prefix or class map, update your styles to match.
